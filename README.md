Here is the Golden HUD V3, an optimized and customizable hud.

|Characteristics|
|---------------|
|New ! Compatibility for SVMod|
|New ! Compatibility for VCMod|
|New ! Wanted Hud|
|New ! Speedometer|
|New ! Notifications|
